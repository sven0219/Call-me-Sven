## ./career-log

- `2018 -> 2020+` 毕业后进入软件行业，从 Java 开发转到运维工程师。这个转向影响了我后来的工程观：代码只有能稳定跑在生产里，才算真正完成。

- `2022 -> now` 目前在一家外企 IT 部门工作，更多关注云平台、交付流水线和平台可靠性。

## ./principles

- 把无聊路径自动化，把奇怪路径文档化。
- 可理解的基础设施，比聪明的基础设施更重要。
- CI/CD 是一个产品，不是一堆 YAML。
- 生产环境要无聊，学习可以怪一点。
- 故障复盘要趁记忆还热的时候写。
